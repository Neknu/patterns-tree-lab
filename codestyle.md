### Naming Conventions
- `CamelCase`:
    - Types (start with capitals): `MyClass`;
    - Functions (start with lower case): `myMethod`;
- `snake_case`:
    - Variables: `my_variable`;
    
- Constants are all capital: `const int SIZE = 10;`

- Use `func() {` on the same line of code
